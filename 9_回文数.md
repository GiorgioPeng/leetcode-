# 思路：  
负数和最后一位是0的数（0除外）肯定不存在回文  
转化成字符串分割成数组再反转然后再转化成字符串
```

/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
    if(x<0||x%10==0&&x!=0)
        return false;
    if(x.toString()==x.toString().split('').reverse().join(''))
        return true;
    else 
        return false;
};
```
